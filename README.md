# MarkTan.ai Daily Dashboard

A static public dashboard for `marktan.ai`.

## Files

- `index.html` — page structure
- `styles.css` — responsive visual design
- `app.js` — weather, clocks, market cards, and curated feed links

## Deploy

Upload these three files to your web host's root directory for `marktan.ai`.

For Vercel/Netlify/Cloudflare Pages, drag the folder into a new static site project or connect it to a GitHub repo.

## Notes

Market quotes use Yahoo Finance chart endpoints and may be delayed or blocked by some browser/network settings. Weather uses Open-Meteo and requires no API key.
